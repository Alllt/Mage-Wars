How to Install the Spellbook Builder Dll for the Mage Wars module on OCTGN

1. After you have downloaded the current release of the dll, see the note below before you start this download.

2. Copy it to the to <InstallPath>\OCTGN\Plugins\MWPlugin folder (create the MWPlugin folder if needed)

3. You will need to check to see if the dll is blocked from executing and if so you will need to "Unblock" the file

4. You can do this by right-clicking on the dll and selecting properties. If you see a button in the lower left hand corner labeled Unblock, click it to unblock the file.

Notes:

The dll is installed into the <InstallPath>\OCTGN\GameDatabase\9acef3d0-efa8-4d3f-a10c-54812baecdda\Plugins\Octgn.MagewarsPlugin folder when you install (or update) the Mage Wars for OCTGN Module.

When a new release is downloaded the dll in this location is updated but not the dll in the <InstallPath>\OCTGN\Plugins\MWPlugin

You can copy the dll to the <InstallPath>\OCTGN\Plugins\MWPlugin folder from the <InstallPath>\OCTGN\GameDatabase\9acef3d0-efa8-4d3f-a10c-54812baecdda\Plugins\Octgn.MagewarsPlugin and start with step 3 above.
